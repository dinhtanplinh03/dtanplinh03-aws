### Week 4 Objectives:

* Understand Amazon RDS and managed database services on AWS.
* Learn how to create, configure, and connect to relational databases in the cloud.

### Week 4 Achievements:

* Developed a solid understanding of AWS managed database services and Amazon RDS architecture.

* Learned the key concepts of Amazon RDS, including:

  * Database Instances
  * Database Engines
  * Storage Allocation
  * Backups and Snapshots
  * Security Groups

* Successfully created and configured an Amazon RDS MySQL database instance.

* Practiced connecting to cloud databases using external database management tools.

* Performed database operations including:

  * Creating databases
  * Creating tables
  * Inserting records
  * Updating data
  * Querying information

* Learned how Amazon RDS simplifies database administration by automating:

  * Backups
  * Software patching
  * Monitoring
  * Maintenance

* Successfully created and managed database snapshots for backup and recovery purposes.

* Gained practical experience working with relational databases in a cloud environment and understanding how applications can securely connect to Amazon RDS.
