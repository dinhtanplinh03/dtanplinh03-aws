### Week 2 Achievements:
* Gained a solid understanding of AWS Identity and Access Management (IAM) concepts, including:

  * Users
  * Groups
  * Roles
  * Policies

* Understood AWS security best practices and the importance of securing cloud resources.

* Learned how to:

  * Create and manage IAM users
  * Create and manage IAM groups
  * Assign permissions through policies
  * Implement access control for AWS resources

* Successfully practiced IAM operations, including:

  * Creating users and groups
  * Assigning users to groups
  * Attaching managed policies
  * Creating custom policies
  * Creating IAM roles
  * Testing and validating access permissions

* Applied the Principle of Least Privilege to ensure users receive only the permissions required for their tasks.

* Developed the ability to manage AWS account security more effectively and follow recommended AWS security practices.
