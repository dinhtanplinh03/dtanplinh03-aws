### Week 7 Achievements:

* Developed a strong understanding of high availability and scalability concepts in cloud environments.

* Learned the core components of Elastic Load Balancing, including:

  * Application Load Balancer (ALB)
  * Target Groups
  * Health Checks
  * Traffic Routing

* Successfully configured an Application Load Balancer to distribute incoming traffic across multiple EC2 instances.

* Gained practical experience with health monitoring and automatic traffic redirection when instances become unavailable.

* Learned the principles of Auto Scaling and how AWS automatically adjusts compute resources based on workload demands.

* Successfully created and configured Auto Scaling Groups with scaling policies to:

  * Increase capacity during high traffic periods
  * Reduce capacity during low traffic periods
  * Improve application availability and cost efficiency

* Built a highly available architecture capable of handling traffic fluctuations while maintaining application performance.

* Developed practical skills in designing scalable AWS infrastructures that follow cloud best practices for reliability and fault tolerance.
