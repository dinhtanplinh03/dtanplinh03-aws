### Week 6 Achievements:

* Developed a solid understanding of AWS monitoring and observability services.

* Learned the core components of Amazon CloudWatch, including:

  * Metrics
  * Logs
  * Alarms
  * Dashboards
  * Events

* Successfully monitored AWS resources and analyzed performance indicators such as:

  * CPU Utilization
  * Network Traffic
  * Disk Activity
  * Database Performance Metrics

* Configured CloudWatch Logs to collect and review system and application logs from AWS resources.

* Created CloudWatch Alarms to detect abnormal resource usage and support proactive system monitoring.

* Built custom CloudWatch Dashboards to visualize infrastructure performance in a centralized view.

* Gained practical experience in monitoring cloud environments, identifying performance issues, and improving operational visibility across AWS services.

* Developed the ability to use CloudWatch as a centralized monitoring solution for AWS infrastructure and applications.
