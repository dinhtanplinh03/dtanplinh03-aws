This section lists the translated blog posts and technical shares focused on game development infrastructure on AWS:

###  [Blog 1 - Game Development Support with AWS Game Dev Toolkit](3.1-Blog1/)
This post introduces how the open-source **AWS Cloud Game Development Toolkit** helps game studios quickly stand up collaboration environments on the cloud. You will learn about deploying the **Perforce P4** version control system and **Unreal Engine Horde** CI/CD specialized workflow using Terraform & Packer templates, slashing setup times from weeks to hours.

###  [Blog 2 - Understanding the Multi-Build Solution on Amazon GameLift Servers](3.2-Blog2/)
This article dives into the **Multi-Build solution on Amazon GameLift**, which allows running multiple game server versions simultaneously on a single fleet. This optimizes operational costs, eliminates fleet deployment overhead, and accelerates rapid iteration testing during Alpha/Beta phases.

###  [Blog 3 - [AWS Tech Share] Transforming Spectators into Players: Building Interactive Gaming Experiences with GameLift Streams](3.3-Blog3/)
This post shares the architecture behind combining **Amazon GameLift Streams** (WebRTC low-latency streaming), **Amazon IVS** (subsecond streaming), and **AWS AppSync** (WebSocket APIs for state synchronization) to convert passive game stream viewing into real-time, two-way interactive gaming experiences directly in browsers.
