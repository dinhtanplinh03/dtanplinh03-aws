###  [Blog 1 - SUPPORT GAME DEVELOPMENT WITH AWS CLOUD GAME DEVELOPMENT TOOLKIT](3.1-Blog1/)
This blog post introduces the AWS Cloud Game Development Toolkit, an open-source toolkit that helps game studios quickly build development environments on AWS. The toolkit provides ready-made solutions for source code management with Perforce, build automation and CI/CD with Unreal Engine Horde, and infrastructure deployed using Infrastructure as Code. This allows development teams to accelerate workflows, optimize operating costs, and focus more on product development rather than infrastructure management.

###  [Blog 2 - CONVERT VIEWERS INTO PLAYERS WITH AMAZON GAMELIFT STREAMS](3.2-Blog2/)
This blog post introduces Amazon GameLift Streams, a solution that integrates with Amazon IVS and AWS AppSync to build real-time interactive gaming experiences. Instead of passively watching streams, users can participate by controlling the game, sending feedback, and interacting directly with the game in their browser with very low latency. This solution optimizes the playtesting process, increases community engagement, and opens up many new forms of gaming experience for game development studios.

###  [Blog 3 - ACCELERATE GAME DEVELOPMENT WITH MULTI-BUILD ON AMAZON GAMELIFT SERVERS](3.3-Blog3/)
This blog post introduces Amazon GameLift Servers' Multi-Build feature, which allows hosting and running multiple game server instances on the same fleet. Instead of redeploying infrastructure for each update, developers can upload a new build to Amazon S3 and select the instance to use when creating a game session. This solution shortens deployment time, supports more efficient Alpha/Beta testing, and accelerates game development through flexible management of multiple server instances.
