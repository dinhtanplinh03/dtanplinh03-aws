### [Blog 1 - HỖ TRỢ PHÁT TRIỂN GAME VỚI AWS CLOUD GAME DEVELOPMENT TOOLKIT](3.1-Blog1/)

Bài viết này giới thiệu **AWS Cloud Game Development Toolkit**, một bộ công cụ mã nguồn mở giúp các studio game nhanh chóng xây dựng môi trường phát triển trên AWS. Bộ công cụ cung cấp các giải pháp sẵn có cho việc quản lý mã nguồn với **Perforce**, tự động hóa quá trình Build và CI/CD bằng **Unreal Engine Horde**, cùng hạ tầng được triển khai theo mô hình **Infrastructure as Code (IaC)**. Nhờ đó, các nhóm phát triển có thể tăng tốc quy trình làm việc, tối ưu chi phí vận hành và tập trung nhiều hơn vào việc phát triển sản phẩm thay vì quản lý hạ tầng.

### [Blog 2 - BIẾN NGƯỜI XEM THÀNH NGƯỜI CHƠI VỚI AMAZON GAMELIFT STREAMS](3.2-Blog2/)

Bài viết này giới thiệu **Amazon GameLift Streams**, một giải pháp kết hợp **Amazon IVS** và **AWS AppSync** để xây dựng trải nghiệm chơi game tương tác theo thời gian thực. Thay vì chỉ xem livestream một cách thụ động, người dùng có thể trực tiếp điều khiển trò chơi, gửi phản hồi và tương tác với game ngay trên trình duyệt với độ trễ rất thấp. Giải pháp này giúp tối ưu quá trình Playtest, tăng cường sự gắn kết của cộng đồng và mở ra nhiều hình thức trải nghiệm game mới dành cho các studio phát triển trò chơi.

### [Blog 3 - TĂNG TỐC PHÁT TRIỂN GAME VỚI MULTI-BUILD TRÊN AMAZON GAMELIFT SERVERS](3.3-Blog3/)

Bài viết này giới thiệu tính năng **Multi-Build** của **Amazon GameLift Servers**, cho phép lưu trữ và vận hành nhiều phiên bản Game Server trên cùng một Fleet. Thay vì phải triển khai lại toàn bộ hạ tầng mỗi khi có phiên bản mới, nhà phát triển chỉ cần tải bản Build lên **Amazon S3** và lựa chọn phiên bản cần sử dụng khi tạo **Game Session**. Giải pháp này giúp rút ngắn thời gian triển khai, hỗ trợ kiểm thử Alpha/Beta hiệu quả hơn và tăng tốc quá trình phát triển game thông qua việc quản lý linh hoạt nhiều phiên bản Game Server.
